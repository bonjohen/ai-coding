Sources

Source inventory for the SQL Competence framework.

Standards & Official Documentation

SQL standards and official database documentation.

ISO/IEC 9075 SQL Standard

ISO/IEC — ISO, 2023

The international standard for SQL, defining the language's syntax, semantics, and conformance requirements.

Relevance: Authoritative reference for SQL syntax and behavior across all levels.

PostgreSQL Documentation

PostgreSQL Global Development Group — PostgreSQL, 2025

Comprehensive documentation for PostgreSQL, covering SQL syntax, administration, performance tuning, and internals.

Relevance: Primary dialect reference used throughout the framework examples.

Books & Deep References

In-depth technical books on SQL, databases, and data systems.

Use The Index, Luke!

Markus Winand — Self-published, 2012

A guide to database performance for developers, focused on SQL indexing and execution plan analysis.

Relevance: Core reference for Levels 3-4 performance optimization and index strategy.

Database Internals

Alex Petrov — O'Reilly Media, 2019

A deep dive into database storage engines, distributed systems, and internal algorithms.

Relevance: Key reference for Level 4-5 concepts around storage, replication, and distributed architectures.

Designing Data-Intensive Applications

Martin Kleppmann — O'Reilly Media, 2017

Comprehensive guide to the principles behind reliable, scalable, and maintainable data systems.

Relevance: Foundation for Level 5 distributed data architecture and consistency trade-offs.

High Performance MySQL

Silvia Botros, Jeremy Tinley — O'Reilly Media, 2021

Practical guide to MySQL optimization, covering schema design, indexing, query optimization, and replication.

Relevance: MySQL-specific depth for Levels 3-4 performance and schema design.

Practitioner Resources

Tutorials, blogs, and community resources for SQL practitioners.

SQL Style Guide

Simon Holywell — sqlstyle.guide, 2016

A consistent style guide for writing clean, readable SQL based on Joe Celko's conventions.

Relevance: Reference for Collaboration & Standards dimension across all levels.

Modern SQL

Markus Winand — modern-sql.com, 2025

Coverage of modern SQL features including window functions, CTEs, lateral joins, and other post-SQL:1999 additions.

Relevance: Key resource for Level 2-3 analytical techniques and modern query patterns.
