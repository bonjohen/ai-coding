Example Projects

Step-by-step walkthroughs demonstrating Claude Code competence at each level.

Level 1: Operator

L1 10 minutes

Writing a String Utility Function

Use Claude Code to generate a string utility function, then review and accept the output.
L1 15 minutes

Finding and Fixing a Bug

Use Claude Code to identify and fix a bug in an existing function, then verify the fix works.

Level 2: Structured Collaborator

L2 45 minutes

Implementing a Feature Across Multiple Files

Decompose a feature request into focused prompts and implement it across model, service, and API layers.
L2 40 minutes

Refactoring a Module with Constraints

Guide Claude Code through a refactoring that preserves the public API while improving internal structure and adding tests.

Level 3: Environment Builder

L3 30 minutes

Setting Up CLAUDE.md, Hooks, and MCP for a New Project

Configure a complete Claude Code environment with layered context files, automated hooks, and an MCP server integration.
L3 35 minutes

Configuring an Environment for Team Onboarding

Set up a project environment so a new team member can be productive with Claude Code immediately without verbal instructions.

Level 4: Workflow Engineer

L4 60 minutes

Running Multiple Claude Code Agents on Parallel Worktrees

Orchestrate multiple Claude Code instances working on independent tasks using git worktrees for isolation.
L4 50 minutes

Adding Claude Code to a CI/CD Pipeline for Automated PR Review

Integrate Claude Code into a GitHub Actions workflow to automatically review pull requests and post feedback.

Level 5: Agentic Systems Expert

L5 90 minutes

Designing an AI Coding Governance Policy

Create a comprehensive governance framework for AI-generated code covering review processes, audit trails, risk tiers, and safety boundaries.
L5 120 minutes

Creating a Phased Adoption Plan for an Engineering Organization

Design a complete adoption strategy for introducing Claude Code to a 100-person engineering organization, covering training, tooling, governance, and measurement.
