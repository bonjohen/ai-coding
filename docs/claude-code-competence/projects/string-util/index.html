Writing a String Utility Function

Use Claude Code to generate a string utility function, then review and accept the output.

Scenario

You need a slugify function that converts arbitrary strings into URL-safe slugs. You want Claude Code to write it for you, and you need to review the result before accepting it into your codebase.

Launch Claude Code and Write Your Prompt

Open your terminal in the project directory and launch Claude Code. You will write a simple prompt asking for a slugify utility function.

At this level, the goal is to get comfortable expressing what you need in natural language. You do not need a perfect prompt — just a clear one.

claude

> Write a TypeScript function called slugify in src/utils/string.ts that converts
> a string into a URL-safe slug. It should lowercase the input, replace spaces
> with hyphens, remove non-alphanumeric characters (except hyphens), and
> collapse multiple hyphens into one.

Review the Generated Diff

Claude Code will present a diff showing the new file it wants to create. Read the diff line by line before accepting. Look for: does the function handle empty strings? Does it trim leading and trailing hyphens? Does the logic match what you asked for?

This is the most important habit to build at Level 1. Never accept a diff without reading it. Even if the code looks right at a glance, read every line.

// Claude Code will show something like:
// src/utils/string.ts
// + export function slugify(input: string): string {
// +   return input
// +     .toLowerCase()
// +     .trim()
// +     .replace(/[^a-z0-9\s-]/g, '')
// +     .replace(/[\s]+/g, '-')
// +     .replace(/-+/g, '-')
// +     .replace(/^-|-$/g, '');
// + }

Test the Function Manually

After accepting the diff, verify the function works by running it. You can ask Claude Code to help you test it, or run a quick check yourself.

At Level 1, even a manual test counts. The point is to verify rather than blindly trust.

npx ts-node -e "
import { slugify } from './src/utils/string';
console.log(slugify('Hello World!'));        // hello-world
console.log(slugify('  Multiple   Spaces ')); // multiple-spaces
console.log(slugify('Special @#$ chars'));    // special-chars
console.log(slugify(''));                     // (empty string)
"
