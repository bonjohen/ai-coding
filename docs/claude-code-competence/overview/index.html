Claude Code Competence Framework

A five-level model of proficiency measured across seven cross-cutting dimensions, from basic tool operation to agentic systems design.

This framework defines competence not as a checklist of features learned, but as measurable growth across seven dimensions that each capture a distinct aspect of effective AI-assisted development.

Every practitioner — from someone running their first prompt to someone designing enterprise-scale governance — operates across all seven dimensions simultaneously. The levels describe how deeply and consistently that capability is expressed.

The Seven Dimensions

Each dimension is independent: a practitioner can be strong in Task Framing but weak in Verification Discipline. The matrix page shows exactly how each dimension is expected to manifest at each level.

The Five Levels

Each level represents a recognizable, observable mode of working with Claude Code. Movement between levels is not linear mastery of isolated skills — it is a shift in how all seven dimensions are engaged together.

Level 1

Operator

Uses Claude Code as a command-line tool with basic prompting.
Level 2

Structured Collaborator

Structures interactions with context, constraints, and verification.
Level 3

Environment Builder

Configures persistent environments with CLAUDE.md, hooks, and MCP.
Level 4

Workflow Engineer

Designs multi-agent workflows and CI/CD integration.
Level 5

Agentic Systems Expert

Architects governance, safety, and organizational AI coding strategy.

How to Use This Framework

Use the Self-Assessment to score yourself across all seven dimensions and get an estimated level. Use the Competency Matrix to see how your weakest dimensions compare to your strongest.

Use the level detail pages for graduation criteria and evaluation tasks. Use the Study Guide to generate a prioritized reading list based on your assessment results.
